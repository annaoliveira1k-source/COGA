COGA — Painel (GitHub Pages)

1) Suba estes arquivos na raiz do repositório:
   - index.html
   - style.css

2) Ative o GitHub Pages:
   Settings → Pages → Deploy from a branch → main → /(root)

3) Links do Notion:
   No index.html, troque os href dos botões pelos links de cada base.
   (Clientes/Demandas/Rotinas/Implantação)

Observação:
- Para outras pessoas acessarem o Notion, publique em Share → Publish to web.
- Se for só para você (logada), não precisa publicar.
